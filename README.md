# 用法

* 'videoId': 影片網址IS
* 'startSeconds': 起始秒數
* 'endSeconds': 結束秒數 
* 'suggestedQuality': 畫質(可以參考 [這裡](https://developers.google.com/youtube/js_api_reference#Playback_quality).)
